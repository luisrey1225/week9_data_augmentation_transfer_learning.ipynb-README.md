# Week 9 – Data Augmentation y Transfer Learning

## Dataset
Se utilizó CIFAR-10, un conjunto de imágenes clasificadas en 10 categorías.

## Comparación
Se compararon tres modelos:

1. Modelo base.
2. Modelo con Data Augmentation.
3. Modelo con Transfer Learning usando MobileNetV2.

## Resultado principal
La comparación permitió observar el impacto de Data Augmentation y Transfer Learning en el desempeño del modelo mediante métricas de accuracy y loss.

Abrir el notebook en Google Colab y ejecutar todas las celdas en orden con la opción Run all.
